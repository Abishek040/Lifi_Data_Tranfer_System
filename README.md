# 💡 LiFi Data Transfer System using Morse Code

## 📌 Project Overview
This project demonstrates a simple and cost-effective LiFi (Light Fidelity) system using a smartphone torch as the transmitter and an LDR sensor as the receiver.

---

## ⚙️ Components Used
- Smartphone (Morse Code Torch App)
- LDR Sensor Module
- Arduino Nano
- 12V Adapter
- Buck Converter
- Reset Button
- Breadboard

---

## 🔁 Working Principle
- Morse code app converts text into light signals
- Smartphone torch transmits ON/OFF light
- LDR detects light intensity changes
- Arduino processes and decodes signals
